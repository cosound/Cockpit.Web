Text Format

  • Bold - {{b|text}}
  • Italic - {{i|text}}
  • Underlined - {{i|text}}
  • Strikethrough - {{s|text}}
  • Marked - {{mark|text}}
  • Tiny text - {{tiny|text}}
  • Small text - {{small|text}}
  • Large text - {{large|text}}
  • Colored - {{color|red|text}}
  • Styled - {{style|color: yellow; font-size: 20px|text}}
  • Link - {{url|http://www.google.com|My link}}
  • New line - {{n}}
